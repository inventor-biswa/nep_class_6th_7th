Phase 1 — Thinking & Logic / सोचना और तर्क

Day 1–3 · Scratch · Input → Process → Output

▼
Day 1
How Machines Think
मशीनें कैसे सोचती हैं?
ConceptActivity
Human Robot game — students give and follow instructions to experience Input → Process → Output. Worksheet: draw your machine flow.
Day 2
Patterns & Sequences
क्रम और पैटर्न
ScratchActivity
Move a character through a maze using sequence blocks. No prior coding needed. Challenge: add sound on finish.
Day 3
Decisions — If/Else
अगर/नहीं तो
ScratchConcept
Conditional thinking: "If button pressed → sprite jumps". Students design a decision tree on paper first, then code it.
💡

Phase 2 — Simulation World / अनुकरण की दुनिया

Day 4–6 · Tinkercad · Virtual circuits

▼
Day 4
Electricity Basics
बिजली की मूल बातें
TinkercadCircuit Loop
Build a virtual LED circuit. Battery → wire → resistor → LED. Understand that a circuit is a complete loop.
Day 5
Control with a Switch
स्विच से नियंत्रण
TinkercadActivity
Add a push button: pressed → LED ON, released → OFF. Switch = decision maker in a circuit.
Day 6
Sensors — Machines See
सेंसर — मशीनें देखती हैं
TinkercadLDR Sensor
Simulate an LDR: dark → LED ON, bright → OFF. Discussion: streetlights, phone auto-brightness.
🔌

Phase 3 — Real Hardware / असली सर्किट

Day 7–8 · Arduino UNO + mBlock

▼
Day 7
First Real Circuit!
पहला असली सर्किट!
ArduinomBlock
Guided build: LED blinks on breadboard. Roles: Builder, Thinker, Presenter. First hardware "Hello World"!
Day 8
Button + LED System
बटन + एलईडी सिस्टम
ArduinoActivity
Add a physical push button with mBlock code. Students modify blink speed and explore freely.
🚀

Phase 4 — Mini Project / लघु परियोजना

Day 9–10 · Build + Present + Certificate

▼
🌙

Smart Night Light / स्मार्ट नाइट लाइट

LDR sensor auto-turns LED on when dark, off when bright — just like real streetlights!

LDR Sensor mBlock Code LED Output Cardboard Housing
Day 9
Build Day
बनाने का दिन
ProjectHardware
Teams assemble Smart Night Light. Each student fills a build log worksheet. Decorate housing with craft materials.
Day 10
Demo & Certificate
प्रदर्शनी और प्रमाणपत्र
ShowcaseCertificate
Each group demonstrates and explains their project. Certificate awarded. Photos for portfolio.

Focus: Faster ramp-up → Control Systems → Actuators → Real-world automation. Building on Class 6.

⚡

Phase 1 — Quick Recap + Jump In / त्वरित दोहराव

Day 1–2 · mBlock from Day 1

▼
Day 1
Recap + mBlock Intro
दोहराव + mBlock परिचय
mBlockReview
Flashcard review of I/O concepts. Direct mBlock intro — LED blink in 10 min. Challenge: blink SOS (3 short, 3 long).
Day 2
Multi-LED: Traffic Light
ट्रैफिक लाइट
mBlockArduino
Control 3 LEDs independently. Build traffic light sequence: Red → Orange → Green. Introduce timing loops and variables.
🎛️

Phase 2 — Control Systems / नियंत्रण प्रणाली

Day 3–5 · Sensors + Conditions + AND/OR Logic

▼
Day 3
Sensor-Based Control
सेंसर आधारित नियंत्रण
LDR / ButtonAutomation
Sensor reads environment → triggers LED or buzzer. Map: What condition? → What action? Real examples: AC thermostat.
Day 4
Temperature Alert System
तापमान अलर्ट
Temp SensorActivity
If temp > threshold → buzzer ON + Red LED. Else → Green LED. Groups set their own threshold values.
Day 5
AND / OR Logic
AND / OR तर्क
Logic GatesmBlock
Multiple conditions: "If dark AND button pressed → alarm". Students design and test their own compound condition.
🤖

Phase 3 — Actuators / एक्चुएटर

Day 6–7 · Servo Motor + Ultrasonic Sensor

▼
Day 6
Servo Motor — Things Move!
सर्वो मोटर — चीजें हिलती हैं!
Servo SG90Activity
Control servo angle 0°→90°→180°. Build a paper arm that waves. Where are servos used? (RC cars, robots, gimbal cameras)
Day 7
Ultrasonic Distance Sense
दूरी नापना
HC-SR04Distance
HC-SR04 measures distance. If object <10cm → buzzer beep. Real-world: parking sensors in cars!
🔗

Phase 4 — System Building / सिस्टम निर्माण

Day 8 · Combine sensor + actuator

▼
Day 8
Design Your Own System
अपना सिस्टम बनाएँ
Full KitSystem DesignActivity
Students draw: "What is my input? Output? Condition?" Then build a partial prototype. Groups share system diagrams. This feeds directly into the final project.
🚀

Day 9–10 · Build + Demo + Certificate

Smart Parking System / स्मार्ट पार्किंग सिस्टम

Ultrasonic detects car → servo opens gate → LED shows status. All Phase 3 components in one real-world model.

HC-SR04 Sensor Servo Gate RGB LED Status mBlock Code Cardboard Model

Build the Parking System
पार्किंग सिस्टम बनाएँ

Teams build and code. Instructor provides guided troubleshooting checklist. Groups may add extras (buzzer, counter).

Demo, Explain & Certify
प्रदर्शनी और प्रमाणपत्र

Demo to teachers/parents. Each student explains their role + one thing they learned. Certificate + project photo for portfolio.

Assessment — NEP Aligned / मूल्यांकन

No written tests · Observation + Demonstration + Portfolio

Cognitive / संज्ञानात्मक

  • Understands I/O logic
  • Can explain their system
  • Identifies real-world uses

Psychomotor / क्रियात्मक

  • Builds circuit independently
  • Handles components safely
  • Debugs a basic issue

Affective / भावात्मक

  • Shows curiosity
  • Collaborates in team
  • Persists when stuck

Methods / विधि

  • Instructor checklist (per day)
  • Build log worksheet
  • Final project demo
